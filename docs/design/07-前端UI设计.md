# 07 - 前端 UI 设计

## 信息层级

```text
Console
  顶栏: 项目名 / 当前用户 / 主题切换 / 安全状态指示
  侧栏:
    - 仪表盘
    - License
    - 客户
    - 产品（第一版只读 1 条）
    - 审计
    - 通知
    - 设置
        - 安全（密钥状态、2FA、密码策略）
        - 偏好
```

## 关键页面

### 登录页 `(auth)/login`

- 步骤：账号密码 -> 2FA TOTP -> 进入。
- 失败次数显示在登录页底部（不写入 URL）。
- 锁定时显示「已锁定，X 分钟后可重试」，不显示具体次数计数器避免暴力探测。

### License 列表 `(dashboard)/licenses`

- 列：客户、产品、状态、签发日期、到期日期、最后心跳、操作。
- 状态使用语义化颜色：active 绿、expired 灰、grace 橙、revoked 红。
- 操作：查看、续期、撤销、复制 Activation Code（已签发未送达时）。
- 筛选：客户、状态、日期范围。

### License 签发 `(dashboard)/licenses` 弹窗 / 子页

- 双模式 Tab：
  - 在线签发：选择客户实例（已注册回调地址） -> 设置有效期 -> 提交 -> Console 直推。
  - 离线签发：粘贴 Cloud ID -> 解析显示（产品、实例、公钥指纹、硬件指纹）-> 设置有效期 -> 提交 -> 显示 Activation Code，点击「复制」并显示「已复制，注意只复制一次」。
- 表单分区：基础信息、Cloud ID 解析（仅离线 Tab）、有效期、备注、确认。

### License 详情 `(dashboard)/licenses/[id]`

- 元数据卡片：状态、客户、产品、Cloud ID 摘要、签发人、签发时间、到期、上次心跳。
- Tab：审计链、心跳记录、Activation Code 历史。
- 操作：续期、撤销、导出。

### 撤销 `(dashboard)/licenses/[id]/revoke`

- 强校验：必须输入「撤销原因」+ 二次确认。
- 显示撤销影响：「在线模式下立即生效；离线模式下记录撤销事件并通知客户，但产品端在到期日前仍可使用。」

### 审计 `(dashboard)/audit`

- 时间倒序流。
- 字段：时间、操作者、动作、目标、结果、链上哈希、签名状态。
- 导出：选定时间段 -> 生成签名压缩包（JSON + 公钥 + 链 hash）。

### 设置 - 安全 `(dashboard)/settings`

- 当前签名密钥状态（A 方案）：是否已加载、最后轮换时间、下次轮换提醒。
- 当前用户的 2FA / 备用码状态。
- 密码策略（只读，由后端配置）。
- 登录历史 + 失败记录。

## 组件库

基于 `shadcn/ui`：

- 通用：Button、Input、Select、Dialog、Sheet、Tabs、Tooltip、Dropdown、Toast。
- 表格：Table（自带排序 / 分页 / 虚拟滚动）。
- 表单：Form、Field、ErrorMessage、ConfirmDialog。
- 业务：CloudIdInput、ActivationCodeDisplay、AuditChainViewer、KeyStatusCard。

## 主题

- 默认跟随系统。
- 支持显式切换：light / dark / system。
- 颜色语义化变量：`--color-success / warning / danger / info`，避免散落硬编码。

## 字体与排版

- 中文：`PingFang SC / Microsoft YaHei / system-ui`。
- 等宽：`JetBrains Mono`（Cloud ID / Activation Code / 哈希展示必须等宽）。
- 行高：1.5；密度高表格场景 1.3。

## 状态指示

- 顶栏右侧「安全状态」徽章：
  - 绿点：密钥已加载、Postgres/Redis 通联。
  - 橙点：可降级运行（如 Webhook 不可达）。
  - 红点：拒绝服务（无密钥、无数据库）。
- 点击徽章打开「系统状态」面板，跳转到 `settings` 安全分区。

## 错误页

- 401 / 403 / 404 / 500 全部使用统一布局；显示 traceid + 一键复制。
- 不暴露后端 stack trace。
