# Mockup

Admin UI 设计稿。计划包含的关键页面：

| 页面                  | 关键元素                                                                  |
|-----------------------|---------------------------------------------------------------------------|
| 登录 / SSO 选择       | username/password 表单 + 可选 SSO 按钮 + 翠绿 #34C759 主按钮              |
| Dashboard             | 总览：活跃 license 数、即将过期、被吊销、心跳异常                          |
| 客户管理 - 列表       | 表格 + 创建客户按钮                                                       |
| 客户管理 - 详情       | 客户基础信息 + 该客户名下所有 license + 签发新 license                    |
| 产品管理              | 产品定义 + features 列表 + 默认套餐                                       |
| License - 列表        | 按客户/产品/状态筛选；每行有：吊销 / 续期 / 下载 `.forge` 文件 按钮       |
| License - 签发        | 选客户 + 产品 + 验证模式 + 颗粒度 + 算法 + 绑定模式 + 过期 → 生成 `.forge` |
| License - 详情        | 完整 payload 字段 + 心跳历史 + 审计                                       |
| 密钥管理              | 列表 / 生成新密钥 / 轮换 / 撤销                                            |
| API Key 管理          | 为 Verifier 签发的 API Key；项目级颗粒                                    |
| 审计日志              | 时间线视图 + 过滤                                                          |
